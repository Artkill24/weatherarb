Methodology
How WeatherArb works
Statistical weather anomaly detection built on 24 years of climatological data.
01
🛰️
ERA5-Land + NASA POWER Baseline
We compute monthly climatological averages from ERA5-Land ECMWF (2000–2024) and validate with NASA POWER satellite data for each city's exact coordinates.
02
📊
Real-time Z-Score Analysis
Every hour, OpenWeatherMap live conditions are compared against the seasonal baseline. Z-Score = (current − historical_mean) / std_deviation. Values above ±2.0 trigger alerts.
03
📡
Free API + Telegram Alerts
All data is publicly accessible via our free REST API. Subscribe to Telegram alerts for your city. No registration, no API key required for the free tier.
